# GitHub Analyzer | محلل GitHub

A simple web tool to analyze GitHub accounts and display comprehensive statistics about repositories and programming languages.
أداة ويب بسيطة لتحليل حساب GitHub وعرض إحصائيات شاملة عن المستودعات واللغات البرمجية.

## ✨ Features | المميزات

* **Display profile information:** (Name, avatar, followers, etc.).
  **عرض معلومات الملف الشخصي:** (الاسم، الصورة، المتابعون...).
* **General statistics:** Number of repositories, stars, and forks.
  **إحصاءات عامة:** عدد المستودعات، النجوم، التفرعات.
* **Language analysis:** Includes visual charts.
  **تحليل اللغات البرمجية:** مع رسم بياني.
* **Repository List:** All repositories with descriptions and topics.
  **قائمة بالمستودعات:** عرض جميع المستودعات مع الوصف والمواضيع.
* **Token Support:** Access private repositories using a GitHub Token.
  **دعم التوكن:** للوصول إلى المستودعات الخاصة.
* **Export Reports:** Export the full report as a Markdown file.
  **تصدير التقارير:** تصدير التقرير بصيغة Markdown.
